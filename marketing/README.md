# Marketing Assets

`marketing/` is for launch and community copy.

Use this directory in two modes:

- current campaign assets at the root, if an active launch is in progress
- versioned historical copy under `marketing/archive/<version>/`

At the moment, the available material is historical:

- `archive/v1.1.0/` — original v1.1.0 launch copy for Twitter/X, Product Hunt, IndieHackers / Reddit, and Chinese communities

Do not reuse archived copy without refreshing:

- version number
- skill / agent / command counts
- command examples
- feature framing
