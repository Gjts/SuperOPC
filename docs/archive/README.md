# Docs Archive

This directory keeps historical design specs, migration plans, and other documents that are useful for understanding how SuperOPC evolved but should not be treated as the current source of truth.

Use these files for:

- architecture history
- migration rationale
- ADR backreferences

Do not use them as the primary usage guide.

Current source of truth lives in:

- `README.md`
- `AGENTS.md`
- `docs/COMMAND-CHEAT-SHEET.md`
- `docs/DIRECTORY-MAP.md`
- the active files under `commands/`, `skills/`, and `agents/`
