Install

5-minute setup.
Any AI tool you use.

Clone once, export to your tool. Works with your existing setup — no subscriptions, no lock-in.

# 1. Clone SuperOPC
git clone https://github.com/gjts/superopc.git
cd superopc

# 2. Export for Cursor
python scripts/convert.py --tool cursor

# 3. Copy to your project
cp -r integrations/cursor/.cursor /your-project/

# 4. Open your project in Cursor — done.
# SuperOPC skills and agents are now active.
Design Principles

Built for one person
doing the work of ten.

1

Sell before you build

Every workflow runs validate-idea first. No code without a real-world signal.

2

Skill-first execution

If a skill exists, use it. Skills encode methodology, not just prompts.

3

TDD is non-negotiable

No production code without a failing test first. Quality is not optional.

4

Eat your own dog food

SuperOPC is built with SuperOPC. Every feature is self-validated.

5

State persists, context survives

The .opc/ directory gives your AI memory that survives across sessions.

6

Advisory over blocking

Guardrails warn, they don't lock you out. You're always in control.
